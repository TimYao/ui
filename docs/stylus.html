全局状态

1、disabled

针对按钮，所以请组合 btn 运用(除了a标签)

add 方式:
(1) class="btn disabled"
(2) 标签属性模式 disabled

2、hide or show

针对display:none控制

add 方式:
class="btn hide"
class="btn show"

3、clearfix

针对清除浮动,但不提倡引入标签class模式，提倡调用clearfix()继承到需要标签上

add 方式:
(1) class="btn clearfix"
(2) .btn clearfix()

4、:link :visited :hover :focus :active

1、a标签目前支持 :hover / :active

add 方式:
class="hover" / class="active" / 默认不填写

add class hover
add class active

2、button 标签目前支持 :hover / :focus / :active / .disabled / [disabled]

add 方式:
class="hover" / class="active"
btn模式下默认为统一色系，无变化，可组合到对应皮肤组件中


5、.active

关于当前项统一.active,包括所有组件下，后期保证该组件下存在,组件会优先该组件下同名属性
